# Designer Skill

Instructions for visual design and user interface layout:
- **Aesthetics**: Build interfaces that feel premium and modern. Use Sleek dark modes, HSL tailored colors, smooth gradients, and glassmorphism.
- **Typography**: Avoid browser default fonts. Use Google Fonts like Inter, Outfit, or Roboto.
- **CSS System**: Implement a consistent CSS token system in `index.css` before writing styles. Focus on utility classes for margins, colors, and font-sizing.
- **Animations**: Incorporate subtle hover effects, active states, and micro-animations to make layouts feel alive.
- **Images**: Use the `generate_image` tool instead of placeholders to create rich, working assets.
