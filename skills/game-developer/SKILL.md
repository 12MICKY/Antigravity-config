# Game Developer Skill

Instructions for game development workflow:
- **Unity Templates**: Custom templates are located in the user home directory (`~/Unity user templates/`).
- **Scripting Languages**: Write C# for Unity, C++ for custom systems, and GDScript for Godot projects. Keep code clean and modular.
- **Optimization**: Prioritize profiling (CPU/GPU) early. Minimize draw calls, optimize asset imports, and manage memory lifecycles carefully.
- **Asset Pipeline**: Automate asset imports, compress textures appropriately for target platforms, and structure resource folders cleanly.
