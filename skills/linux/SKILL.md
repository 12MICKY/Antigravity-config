# Linux Skill

Instructions for Linux environment operations:
- **Systemd**: Deploy and manage user-level services and path-watchers (e.g. copying `.service` / `.path` units to `~/.config/systemd/user/` and reloading daemon).
- **Package Management**: Use Homebrew (`brew`) for macOS/Linux userland packages and `apt` for system package installation.
- **Environment**: Configure global and local environment variables in `~/.zshrc` or project-specific `.env` files.
- **Commands**: Leverage standard Linux utilities (e.g., `awk`, `find`, `sed`, `rsync`) for advanced file manipulation.
