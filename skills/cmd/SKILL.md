# CMD Skill

Instructions for command customization and scripting:
- **Aliases**: Manage customized command aliases in `~/customizecmd/shell/aliases.zsh`.
- **Scripting**: Write shell scripts (`.sh`) using safe bash conventions (e.g. `set -e`, proper variable quoting).
- **PATH Config**: Maintain path additions in `.zshrc` (e.g. adding `~/.local/bin` and Homebrew paths).
- **Execution**: Ensure customized commands and scripts are marked executable using `chmod +x`.
