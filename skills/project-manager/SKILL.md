# Project Manager Skill

Instructions for managing project milestones and task lists:
- **Roadmaps**: Maintain a clear roadmap of planned features (e.g., in `MEMORY.md` or dedicated roadmap files).
- **Task Prioritization**: Prioritize high-impact features and divide work into manageable steps.
- **Issue Tracking**: Write descriptive descriptions for tasks, issues, and PR comments.
- **Milestones**: Track deliverables by setting target timelines and monitoring resource constraints.
