# CLI Skill

Instructions for CLI configuration and navigation:
- **Statusline**: Configure the status line using a custom script. Enable/disable via the `/statusline` slash command or `settings.json`.
- **Keyboard Shortcuts**: Use `Ctrl+D` or `/exit` to close the CLI session.
- **TUI Settings**: Customize color schemes and configuration options in `~/.gemini/antigravity-cli/settings.json`.
- **Diagnostics**: Read and inspect CLI log files under `~/.gemini/antigravity-cli/log/` to troubleshoot failing commands.
